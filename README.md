# Airline Passenger Satisfaction Prediction ✈️

## Project Overview

This project focuses on predicting airline passenger satisfaction using Machine Learning techniques.

The objective is to analyze passenger feedback, travel information, service ratings, and flight details to identify the key factors that influence customer satisfaction.

## Dataset

The dataset contains airline passenger information including:

- Customer demographics
- Travel type
- Flight class
- Service ratings
- Flight delays
- Passenger satisfaction

Total Records: 25,976 passengers

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Data Encoding
4. Feature Selection
5. Machine Learning Model Building
6. Model Evaluation

## Machine Learning Model

Algorithm Used:

- Random Forest Classifier

Model Performance:

- Accuracy: 94.96%

## Key Insights

The analysis identified important factors affecting passenger satisfaction, including:

- Inflight services
- Online boarding experience
- Seat comfort
- Travel class
- Flight delay impact

## Conclusion

The project demonstrates how machine learning can be used to analyse customer experience and predict passenger satisfaction.
